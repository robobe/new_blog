    VSCode python dev environment using uv and ruff

    uv is a fast Python package manager

    install

    1
    curl -Ls https://astral.sh/uv/install.sh | sh

    basic usage

    Create project with venv automatically

    1
    uv init

    Add dependency

    1
    uv add numpy

    add dev dependency

    1
    uv add --dev pytest

    Run script inside project env

    1
    uv run python app.py

    no need to activate venv manually

    Install from lock file

    Reproducible install.

    1
    uv sync

    Lock file

    A lock file freezes the exact versions of all dependencies — including transitive dependencies.

    It guarantees:

    “Everyone installs exactly the same environment.”

    File Purpose
    pyproject.toml What you WANT
    uv.lock What you ACTUALLY GET

    install exactly locked versions

    1
    uv sync

    Transitive Dependencies

    for example install uv add fastapi it install many depend packages

    Lock file records ALL of them

    pre-commit

    pre-commit runs checks automatically before git commit.

    If checks fail → commit is blocked.

    It prevents:

    • Broken formatting
    • Unused imports
    • Simple lint errors
    • Bad whitespace
    • Committing junk

    install

    1
    uv add --dev pre-commit

        args: ["--fix"]
      - id: ruff-format

  - repo: https://github.com/pre-commit/pre-commit-hooks
    rev: v4.6.0
    hooks:
      - id: trailing-whitespace
      - id: end-of-file-fixer
      - id: check-yaml
      - id: check-toml

    Activate and usage

    1
    uv run pre-commit install

    Now it run the pre-commit script every git commit

    Run manually

    1
    uv run pre-commit run --all-files
