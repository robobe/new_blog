    Build CV_Bridge for custom opencv build

    Build opencv 4.10 with cuda support Use Docker to create build image

    Build on docker

    • docker base cuda devel with cudnn FROM nvidia/cuda:12.6.0-cudnn-devel-ubuntu22.04
